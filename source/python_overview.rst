Overview
========

The basic functionality of the C++ code is to model a part (i.e. a *subfield*) of one of the four CCDs of the normal cameras (``1, 2, 3, 4``) or of the fast cameras (``1F, 2F, 3F, 4F``) of one of the cameras on the platform. However, configuring the YAML input file for many different use cases can be a complex and time consuming process. Thus, insetad we offer the possibility to use PlatoSim directly from Python.
