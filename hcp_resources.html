Compression¶

LC only (1 file = 7.4 Mb):

  • 1 cfile : 1.6 Mb

  • 8 cfiles : 13.1 Mb

  • 8 ccfiles: 12.9 Mb

Both large memory nodes satisfy the memory requirement for the “minimum cases” but for the “optimal case” only Superdome satisfy the memory needed to store the files. In any case we should find a way to compress and transfer teh files meanwhile the simulations are running. In conclusion the task will be possible and so within a week from start time of executing the jobs.
