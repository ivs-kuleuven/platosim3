Running Simulations with PlatoSim3

To run simulations with PlatoSim3, you will have to feed one configuration file as input to the simulator, possibly together with a few additional files. Launching simulations can be done either on the command line or in Python.

Note that - before running PlatoSim3 - you must have an environment variable PLATO_PROJECT_HOME, set to the base folder of PlatoSim3, as described here.

Command Line

To initiate a simulation on the command line, cd to the /build directory and type:

1 ./platosim <input file> <non-existing output file> [<log file>]

Python

Alternatively, you can make use of our Python wrapper to configure and run PlatoSim3. How to do this and how to set up you Python environment is described here.

Under /docs/tutorials you can find a number of tutorials (in the form of Jupyter Notebooks) to get you going. Have a look here to find out how to configure your Python environment to be able to get started.
