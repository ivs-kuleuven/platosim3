Installation
============

.. attention::

   Note that if you are using the OpenBLAS library then BLAS and occationally LAPACK becomes redundant. Thus, several prerequites may be needed to succesfully install the PLATO LESIA repository containing the script of the provisional L1-pipeline. Please have a look at the :doc:`Troubleshooting <l1_troubleshooting>` page.

   
1. The LESIA repository is a SVN (subversion) repository. Hence first install svn (using ``sudo apt install <subversion>``). Create a directory called ``plato`` (preferably on your ``$DATA`` storage if installing on a cluster) and enter the directory:

   .. code-block:: shell

      mkdir </path/to/plato>
      cd </path/to/plato>

   Now download the SVN repository by providing your ``username`` and ``password``: 
      
   .. code-block:: shell

      svn checkout https://version-lesia.obspm.fr/repos/PLATO/algorithms/ --username <username> --password <password>

      
2. Prior to the installation you need to run setup script called ``/PLATOnium/setup.sh``, also explained in the general :ref:`step-by-step installation instructions <repo_install_step>`. This scirpt takes in fact two arguments; the frist being ``</path/to/plato_workdir>`` and the second being the path to your newly created folder ``</path/to/plato>``. Thus, simply run:
   
   .. code-block:: shell

      ./setup_plato.sh </path/to/plato_workdir> </path/to/plato>
   
      
3. First make sure that your PLATOnium (Poetry) Python environment is activated, and then install PLATO by:

   .. code-block:: shell

      cd $PLATO/algorithms
      make install

      
.. admonition:: Update PLATO repository

   To update the repository (whenever a new change is made) simply go to ``cd $PLATO/algortihms`` and do:

   .. code-block:: shell

      svn update
      sudo make


      
..
   .. _l1_installation_env:

   Create environment
   ------------------

   1. Currently the only way to ensure that both PlatoSim, PLATOnium, and (LESIA) PLATO are installed altogether properly is to create a python3 environment. If not installed, first install the python3-venv package:

      .. code-block:: shell

	 sudo apt install python3-venv

   2. Next create a folder somewhere on your system of your choice called ``plato``. This will be the placeholder for all the code that will be downloaded and installed for the LESIA PLATO repository.

   3. Create the virtual environment and make sure that your environment is activated before you continue:

      .. code-block:: shell

	 python -m venv $PLATO/lib/python
	 source $PLATO/lib/python/bin/activate

      .. caution::

	 Some times the ``venv`` do not install the python libraries correctly into your ``lib/python`` library. If that happens you can always install the package by first asking ``which pip`` (with the venv activated), and then use this path to install the library. E.g. ``$PLATO/lib/python/bin/pip> install <package>``.

      .. admonition:: Activate environment

	 Remember to activate your virtual environment every time you use this repository:

	 .. code-block:: shell

	    source $PLATO/lib/python/bin/activate
