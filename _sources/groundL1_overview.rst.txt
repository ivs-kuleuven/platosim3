Overview
========

In the following we describe how to create representative P1 sample light curves. The full documentation of how to run PlatoSim imagettes are given in ??. This documentation also shows how to run these simulations on the HPC. The following shows an outline of the reduction steps involved:

* Run simVariability on this input
* Simulate imagettes time series with PlatoSim
* Simulate microscanning sequence and compute inverted PSFs for each star
* Apply jitter and drift correction
* Extract photometry using PSF fitting

.. note::
   Microscanning is needed for each target, each quater, each camera.
