Overview
========

We now offer the choice to either install:

* Pre-built versions of PlatoSim3 via Conda -- :doc:`install_conda`
* Download and build the software yourself -- :doc:`install_source`.

The latter option gives you the freedom to make changes to the code yourself, but it is a more complex procedure and in most cases this is not needed.
