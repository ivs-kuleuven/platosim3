Pre-processing
--------------

The first step in the L1 pipeline is running a so-called pre-processing module. This module, called ``pproc.py``, performs the following corrections:

  - Bias subtraction
  - 

The following mandatory PlatoSim outputs:

  - ``WritePixelMaps``
  - ``WriteSmearingMaps``
  - ``WriteThroughputMaps``
  - ``WriteFlatfieldMap`` (Will be calculated in FOV later)
  - ``WriteStarPositions``
  - ``WriteACS`` (Optional if jitter cadence is below 4 mHz: then use flag ``--high-cadence-displacement``)
  - ``WriteDiffusedPSF`` (if MappedFromFile PSFs are used and if diffusion is included)
  - ``WriteRotatedPSF``  (if MappedFromFile PSFs are used and if diffusion is turned off)
  - ``WriteHighResolutionPSF`` (if analytic non-gaussian PSFs are used)
  - ``WriteStarCatalog``
  - ``WriteTransmissionEfficiency``

.. note::
   The pre-processing scirpt extract the star IDs from the PlatoSim HDF5 ``StarCatalog/starID`` output file entry. The information given in this data set must be consistent with the catalogue your are working and for which you extract the stars for which you  want to generate simulations.

    
Note that after having run all processing modules you can remove the following files:

.. code-block:: shell

   <starID>-pp.hdf5
   <starID>_psfccd.fits
   <starID>_displacement.dat
   <starID>.times
   
they can be quite large in particular if the time-series are long and if needed they can be re-generated with ``pproc.py`` from <starID>.hdf5
The other files generated by pproc.py are anyway small.


On-ground L1 pipeline
---------------------

The strategy for the on-ground photometric extraction is using PSF photometry. This strategy allows higher precision photometry than binary mask extraction, since each star (targets and contaminats) within an imagette can be moduled, which especially is advantagous for PLATO's scarse sampling of the PSF onto the pixel-grid focal plane. This is especially true for crowded regions, however, PSF fitting do also fail when crowding becomes to prominents (e.g. when stars contaminants are within the same pixel as the target star).

The module ``pproc.py`` has the following mandatory PlatoSim outputs:

.. code-block:: shell

   ThroughputMaps
   WriteDiffusedPSF (when analytical MappedFromFile PSFs are used)
   WriteHighResolutionPSF (when analytical non-gaussian PSFs are used)
   WriteTransmissionEfficiency
   WriteStarPositions
   WriteFlatfieldMap
   WriteThroughputMaps
   WriteSmearingMaps
