Getting started¶

All information needed to install and update the PlatoSim3 software on your system is described in the sections Installation and Running PlatoSim. Please, follow these instructions carefully before running the simulator. As part of the PlatoSim software package, a suite of tools are made available which we explain in the Python Tutorials. Particular relevant for simulating multi-camera imagette time series or light curves, the PlatoSim toolkit called PLATOnium may be of interest. In case of problems see our Troubleshooting section. Lastly, for more information about PlatoSim consult the PlatoSim3 Paper and the Architecture.
