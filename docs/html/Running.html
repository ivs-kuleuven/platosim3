Running PlatoSim3

The scheme below shows an overview of the procedure to run the PlatoSim3 software on your system.

Once you have downloaded or updated the software,

If you followed the installation procedure for developers, you have to build it in order to be able to run it. The first time you download the software and on the rare occasions we update the dependencies (i.e. software packages we use in the code and that are distributed together with the PlatoSim3 code), you will have to resolve the dependencies (as described together with the build procedure) .

Please, note there are a number of prerequisites to be able to run PlatoSim3.
