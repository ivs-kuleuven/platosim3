PLATO Simulator  3.0
An end-to-end simulator for the Plato mission.
Tutorials

Under /docs/tutorials you can find a number of Jupyter Notebooks (previously iPython Notebooks), each of which serves as a (documented) tutorial for a specific use case. To view such a tutorial, cd to the /docs/tutorial directory (or one of its sub-directories) and type

jupyter notebook

Open a browser and go to http://localhost:8888/tree (this may happen automatically). This will show the directory structure of the /docs/tutorials. Each of the sub-directories (e.g. FineGuidanceStars) contains (at least) one IPYNB file (and a set of images that are included in such notebooks). By clicking on the name of such an IPYNB file, a notebook will be shown in your browser.

The individual cells of the notebook can be run one-by-one by pressing the Play button in the toolbar or by selecting Cell > Run Cells.

Jupyter

If you have installed Anaconda, you can install Jupyter by typing the following command:

conda install jupyter

Experience has learnt us that this command causes problems in using Spyder (the GUI that comes with Anaconda). This can be solved by executing the following series of commands:

spyder --reset
conda update spyder
conda update matplotlib
conda update pytables
conda update h5py

Launching Jupyter while having Spyder open in the same environment crashed the kernel in Spyder. This can be solved by cloning the platosim environment and launching Jupyter and Spyder in two different environments:

conda create -n platosimjupyter --clone platosim
