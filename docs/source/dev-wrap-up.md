# Wrap-Up{#dev-wrap-up}

After following the instructions on how to install, build, and update PlatoSim3, you should have the following:

- on your personal GitHub page: the forked repository;
- on your local machine: a local copy (i.e. clone) of the repository;
- `upstream` and `origin` in the list of remote repositories.

Additionally, you should be able to:

- resolve the dependencies;
- build the code, after a change in dependencies or in the code itself;
- get the latest version of the software (from the `upstream`) on your local machine
- send the code from your local machine to the `origin`;
- open a pull request to have your changes (reviewed and) incorporated in the `upstream`.
