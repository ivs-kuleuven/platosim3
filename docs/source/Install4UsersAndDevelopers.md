# Installing/Updating PlatoSim3 {#InstallUsersDevelopers}

We now offer the choice to either install pre-built versions of PlatoSim3 @ref ViaConda "via conda" or to download and build the software @ref DownloadUpdateBuild "yourself". The latter option is more complex but gives you the freedom to make changes to the code.
