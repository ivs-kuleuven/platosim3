# Installing/Updating PlatoSim3 {#InstallUsersDevelopers}

We now offer the choice to either install pre-built versions of PlatoSim3 @ref ViaConda "via conda" or to download and build the software @ref DownloadUpdateBuild "yourself". The latter option gives you the freedom to make changes to the code yourself, but it is a more complex procedure and in most cases this is not needed.
