# Wrap-Up{#user-wrap-up}

After following the instructions on how to install update PlatoSim3, you should have the following:

- Python Anaconda installation;
- dedicated Conda environment in which you have installed PlatoSim (make a separate environment for the `master` and `develop` branches).

Additionally, you should be able to:

- activate the Conda environment when you want to start using PlatoSim;
- de-activate the Conda environment when you are done;
- update the PlatoSim installation to the latest version or a specific version.
