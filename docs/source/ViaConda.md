# Installing PlatoSim3 via Conda {#ViaConda}

To make life easier on the people who want to use PlatoSim without ever wanting to touch the code, we have started using <a href="https://jenkins.io/">Jenkins</a> to automatically build PlatoSim, enabling you to download the latest successfully built version or the @ref Branching "master or develop branch", or any specific version(s) of these. The flowchart below summarises the steps you have to take.

To be able to @ref InstallViaConda "install" PlatoSim3 via conda, have a look at the @ref ReqsInstallViaConda "prerequisites" first.
