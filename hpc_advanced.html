Multi-threading (GPU)¶

Unfortunately, PlatoSim3 do not support multithreading so this is just extra knowledge. If OpenMPI, use the -L task value to execute a job on multiple nodes. This can be combined with -L lproces which allows multithreading (i.e. CPUs per task) #PBS -L task=1:lproces=1. Intel-based toolchains (e.g. intel/2018a, offering Intel compilers and MKL libraries)
